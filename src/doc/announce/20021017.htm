The Netlabs EPM Distribution Project launches it's website

The new project of netlabs.org, named the 'Netlabs EPM Distribution' (NEPMD), hast just launched its new website at nepmd.netlabs.org.

The project team invites you to visit the site and learn about the benfits and new functionality, that this distribution provides to users of OS/2 and eComStation. All efforts target the developers as well as plain endusers - if you don't know, how EPM can do for you, you have missed a lot ! In opposite to the EPM normally shipped with OS/2 and eComStation, the Netlabs EPM Distribution intends to make installation, configuration and usage of additions to the EPM so easy that every average user can use and configure all of them. The first version of the Netlabs EPM Distribution may not achieve these goals in all aspects already, but it is already a huge step forward and an important first milestone on the road ahead.

See also the project team holding two sessions on the Netlabs EPM Distribution at Warpstock Europe 2002!

Das Netlabs EPM Distribution Projekt öffnet seine Weibseiten

Das neue Projekt von netlabs.org, die 'Netlabs EPM Distribution' (NEPMD), hat soeben seine Webseiten auf nepmd.netlabs.org erröffnet.

Das Projektteam lädt Sie herzlich dazu ein, auf diesen Seiten vorbeizuschauen, um mehr über die Vorteile und neuen Funktionen zu erfahren, die diese Distribution OS/2 und eComStatiuon Benutzern bietet. Alle Erweiterungen helfen sowohl Entwicklern als auch Endbenutzern - wenn Sie den EPM noch nicht benutzt haben, haben Sie bestimmt eine Menge verpasst! Im Gegensatz zum EPM, der mit OS/2 und eComStation ausgeliefert wird, will die Netlabs EPM Distribution die Installation, Konfiguration und das Hinzufügen von Erweiterungen so einfach machen, dass jeder Endbenutzer die Installation, Konfiguration und das Hinzufügen von Erweiterungen problemlos bewältigen kann.

Sehen Sie auch das Prjektteam in zwei Vorträgen über die Netlabs EPM Distribution auf Warpstock Europa 2002!
